# Pudge
> 👹 Pudge[pʌdʒ], Dota2中热门英雄，Gank类型中的佼佼者，集爆发、控制、输出、坦克于一身！

## 简介

**适宜人群** 本身已经或者打算使用微信小程序开发，且需要运行在多端小程序的用户（H5、RN请忽略）

**Pudge** 是一套遵循 [微信小程序原生](https://developers.weixin.qq.com/miniprogram/dev/) 语法规范的 **多小程序端开发** 解决方案。现如今小程序端的形态多种多样，微信小程序、支付宝小程序、百度小程序、头条小程序、美团小程序等各种小程序大行其道。当业务要求在一个新的渠道也投放自己的小程序并且需要具备不同的表现形式时，多端开发，多套维护是十分麻烦和苦难的，那么一套代码多端运行就显得极其重要的。

使用 **Pudge**，我们可以只书写一套微信小程序代码，再通过 **Pudge** 的编译工具，将源代码分别编译出可以在不同端（微信/百度/支付宝/头条小程序等）运行的代码。
